# To Do List
In this group project we made a To Do List. We used JavaScript and to style - SASS.

https://podsi08.github.io/To_Do_List/

## Functionality
- add tasks,
- priorityzing tasks,
- marking done tasks,
- sorting tasks,
- removing done/all tasks
